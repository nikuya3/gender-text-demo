Author gender detection

A probabilistic model which attempts to detect the authors' gender using a sample text.

The detection is done using character-level convolutional neural networks.

The neural networks were trained on English blogs, so similar text samples should work the best.

Keep in mind that this demo uses a less accurate prediction model due to resource constraints.
